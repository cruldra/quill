title: 主题
---

主题允许你以最少的努力轻松使编辑器看起来美观。Quill 提供两个官方支持的主题:[Snow](#snow) 和 [Bubble](#bubble)。

### 用法

```html
<!-- 添加主题的样式表 -->
<link rel="stylesheet" href="{{site.cdn}}/quill.bubble.css" />

<script src="{{site.cdn}}/quill.js"></script>
<script>
  const quill = new Quill('#editor', {
    theme: 'bubble', // 在配置中指定主题
  });
</script>
```

### 自定义

主题主要通过其 CSS 样式表控制 Quill 的视觉外观,许多更改可以通过覆盖这些规则轻松完成。与任何其他 Web 应用程序一样,最简单的方法是使用浏览器开发者控制台检查元素以查看影响它们的规则。

许多其他自定义可以通过相应的模块完成。例如,工具栏可能是最明显的用户界面,但大部分自定义是通过[工具栏模块](/docs/modules/toolbar/)完成的。

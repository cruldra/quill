Quill 提供了多种自定义方式以满足你的需求。本节专门介绍如何调整现有功能。有关添加新功能,请参阅[模块](/docs/modules/)部分;有关样式设置,请参阅[主题](/docs/themes/)部分。


## 容器

Quill 需要一个容器来附加编辑器。你可以传入 CSS 选择器或 DOM 对象。

```javascript
const quill = new Quill('#editor');  // 将使用第一个匹配的元素
```

```javascript
const container = document.getElementById('editor');
const quill = new Quill(container);
```

如果容器不为空,Quill 将使用现有内容进行初始化。

## 选项

要配置 Quill,请传入一个选项对象:

<SandpackWithQuillTemplate
  files={{
    "/index.js": `
const options = {
  debug: 'info',
  modules: {
    toolbar: true,
  },
  placeholder: 'Compose an epic...',
  theme: 'snow'
};
const quill = new Quill('#editor', options);
`}}
/>

可识别以下键:

### bounds

默认值: `document.body`

DOM 元素或 DOM 元素的 CSS 选择器,编辑器的 UI 元素(即工具提示等)应限制在其中。目前,它只考虑左右边界。


### debug

默认值: `warn`

[debug](/docs/api/#debug) 的快捷方式。注意 `debug` 是一个静态方法,会影响页面上 Quill 编辑器的其他实例。默认情况下仅启用警告和错误消息。

### formats

默认值: `null`

可识别并可存在于编辑器内容中的格式列表。

默认情况下,允许使用 Quill 库中定义的所有格式。
要将格式限制为较小的列表,请传入要支持的格式名称数组。

你可以使用[注册表](/docs/registries/)创建全新的格式或更全面地自定义内容。
指定 `registry` 选项将忽略此 `formats` 选项。

<Sandpack
  defaultShowPreview
  activeFile="index.js"
  files={{
    'index.html': `
<!-- Include stylesheet -->
<link href="{{site.cdn}}/quill.snow.css" rel="stylesheet" />
<div id="editor">
</div>
<!-- Include the Quill library -->
<script src="{{site.cdn}}/quill.js"></script>
<script src="/index.js"></script>`,
    "/index.js": `
const Parchment = Quill.import('parchment');

const quill = new Quill('#editor', {
  formats: ['italic'],
});

const Delta = Quill.import('delta');
quill.setContents(
  new Delta()
    .insert('Only ')
    .insert('italic', { italic: true })
    .insert(' is allowed. ')
    .insert('Bold', { bold: true })
    .insert(' is not.')
);
`}}
/>

### placeholder

默认值: 无

编辑器为空时显示的占位符文本。

<SandpackWithQuillTemplate
  files={{
    "/index.js": `
const options = {
  placeholder: 'Hello, World!',
  theme: 'snow'
};
const quill = new Quill('#editor', options);
`}}
/>


### readOnly

默认值: `false`

是否将编辑器实例化为只读模式。

<SandpackWithQuillTemplate
  files={{
    "/index.js": `
const options = {
  readOnly: true,
  modules: {
    toolbar: null
  },
  theme: 'snow'
};
const quill = new Quill('#editor', options);
const Delta = Quill.import('delta');
quill.setContents(
  new Delta()
    .insert('Hello, ')
    .insert('World', { bold: true })
    .insert('\\n')
);

`}}
/>

### registry

默认值: `null`

默认情况下,Quill 定义的所有格式都通过编辑器实例之间的共享注册表在编辑器内容中受支持。对于简单的用例,使用 `formats` 来限制格式;对于更大的自定义,使用 `registry`。指定此 `registry` 选项将忽略 `formatting` 选项。了解更多关于[注册表](/docs/registries/)的信息。

### theme

要使用的主题名称。内置选项为 `"bubble"` 或 `"snow"`。无效或假值将加载默认的最小主题。请注意,仍需手动包含主题的特定样式表。有关更多信息,请参阅[主题](/docs/themes/)。

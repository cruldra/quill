Quill 支持多种格式,包括 UI 控件和 API 调用。

默认情况下,所有格式都在 Quill 编辑器中启用和允许。可以使用 [formats](/docs/configuration/#formats) 选项进行配置。这与在[工具栏](/docs/modules/toolbar/)中添加控件是分开的。例如,你可以配置 Quill 允许将粗体内容粘贴到工具栏中没有粗体按钮的编辑器中。

#### 内联格式

- 背景颜色 - `background`
- 粗体 - `bold`
- 颜色 - `color`
- 字体 - `font`
- 内联代码 - `code`
- 斜体 - `italic`
- 链接 - `link`
- 大小 - `size`
- 删除线 - `strike`
- 上标/下标 - `script`
- 下划线 - `underline`

#### 块级格式

- 引用块 - `blockquote`
- 标题 - `header`
- 缩进 - `indent`
- 列表 - `list`
- 文本对齐 - `align`
- 文本方向 - `direction`
- 代码块 - `code-block`

#### 嵌入内容

- 公式 - `formula` (需要 [KaTeX](https://katex.org/))
- 图像 - `image`
- 视频 - `video`
